Calibration

Choosing calibration project

Click desired calibration project from file list; clicked project is opened. Measured-column indicates date of last calibration for that project, and similarly, Elapsed tells time in hours since last calibration. If that time is more than 18 hours, project is shown with bold text to remind about forgotten calibration.

Measuring selected calibration project

Insert standard sample to sample holder, if required for selected calibration. Click Calibrate; wait for results.

Exporting calibration project(s)

As in Project Explorer.
